# Web Scraping with BeautifulSoup

This project demonstrates how to collect structured data from a public website using Python. The scraper extracts book titles, prices, availability status, and ratings from Books to Scrape and stores the data in a CSV file for further analysis. It showcases practical skills in HTML parsing, data extraction, and dataset creation using Requests, BeautifulSoup, and Pandas.


## Project Description

This project demonstrates how to scrape book information from a public website using Python.

The script extracts:

- Book Title
- Price
- Availability
- Rating

The collected data is stored in a CSV file for further analysis.

## Technologies Used

- Python
- BeautifulSoup
- Requests
- Pandas

## Website Used

http://books.toscrape.com/
